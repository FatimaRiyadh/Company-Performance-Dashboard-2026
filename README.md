# Company-Performance-Dashboard-2026

##  Project Overview
A comprehensive, practical project aimed at analyzing the financial and customer service data for the company's branches 
(Baghdad and Basra) for the year 2026 and transforming it into an interactive dashboard that supports decision-making.

## Individual Steps I Took in the Project:

1) Data Preparation: Raw data was cleaned and prepared using Power Query within Power BI to ensure accurate results.
2) Indicators: Formulas were created to measure revenue, profit, and profit margin, in addition to Service Level Indicators (SLAs).

Visual Design:

Cards: To display live financial results.

Charts: A bar chart for sales and a pie chart for ticket statuses.

Controls: Interactive filters for branches (Baghdad/Basra) and time periods.
